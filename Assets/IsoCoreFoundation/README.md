# ISO-Core Foundation Content Root

This root mirrors the useful ISO-CORE content architecture without copying its
assets:

- `Data/` - ScriptableObject definitions and baked content data.
- `Obj/` - original LIT-ISO prefabs and placement/product previews.
- `Graphics/` - original LIT-ISO sprites, icons, atlases, and future normal maps.

Only original or generated LIT-ISO assets belong here. ISO-CORE Playtest files are
metadata references only and must not be imported, copied, or wired into scenes.
