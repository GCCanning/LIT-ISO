TILE TYPES EXPLAINED:

NeighbourTile (14 types)
├─ What: Rules-based tile that checks adjacent cells
├─ Why: Seamless terrain blending (no visible seams at boundaries)
├─ How: RuleTile system checks 8 neighbors (N, S, E, W, NE, NW, SE, SW)
├─ Output: Single sprite from rule-matching
└─ Examples: NeighbourTile_Plains_FlatGrass

RandomTile (15 types)
├─ What: Tile that picks randomly from sprite variants
├─ Why: Visual variety (prevents monotonous repeating tiles)
├─ How: Contains multiple sprites, picks one at placement
├─ Output: Different sprite each placement (deterministic seed)
└─ Examples: RandomTile_Plains_Plants (18 variants)

ColliderTile (15 types)
├─ What: Invisible collision shape for cliff edges
├─ Why: Physical interaction (player can't walk through cliffs)
├─ How: Placed when flat cell is next to raised cell
├─ Output: Polygon/box collider
└─ Examples: tile-border-left, tile-border-right, slope-a

CHUNK GENERATION SEQUENCE:

1. Player Position
        ↓
2. Calculate Chunk Coordinate (32x32 cells)
        ↓
3. Iterate 32x32 cells in chunk:

   For each cell (x, y):

   a) Sample biome via Perlin(x + offset, y + offset)

   b) Sample height via Perlin(x + seed, y + seed)
      → Flat if < threshold
      → Raised if ≥ threshold

   c) Place ground tile
      FLAT:   NeighbourTile_[Biome]_FlatGrass
      RAISED: NeighbourTile_[Biome]_RaisedGrass

   d) If RAISED, place elevation sprite:
      RandomTile_[Biome]_RaisedGrass (visual only)

   e) If RAISED, place cliff colliders:
      Check 4 neighbors (N, S, E, W)
      → If neighbor is FLAT, place tile-border-[direction]

   f) If FLAT and Rand() < decorationChance:
      Place RandomTile_[Biome]_Plants (decoration)

   g) If FLAT and Rand() < spawnChance:
      Instantiate ResourceNode(Oak Tree | Rock)

4. Finalize: CompositeCollider2D merges all colliders

KEY PARAMETERS TO TWEAK:

File: Assets/World/Biomes/BiomeDefinition_Plains.asset

decorationChance: 0.018
└─ Increase → More plants visible (0.04 = 4% per cell)
└─ Decrease → Sparse world (0.01 = 1% per cell)

raisedTileThreshold: 0.68
└─ Increase → More flat terrain (fewer cliffs)
└─ Decrease → More elevated terrain (more cliffs)

heightNoiseScale: 0.08
└─ Increase → Larger terrain features
└─ Decrease → Smaller, more chaotic terrain

File: Assets/World/ResourceNodes/Node_OakTree.asset

spawnChance: 0.06
└─ Increase → More trees visible (0.10 = 10% per cell)
└─ Decrease → Fewer trees (0.03 = 3% per cell)

harvestCooldown: 45
└─ Respawn time in seconds after harvest

minimumSpacing: 2.5
└─ Distance in units between trees (prevents clustering)
